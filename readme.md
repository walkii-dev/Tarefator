## Tarefator

O Tarefator API é uma API Rest que organiza as tarefas dos usuários. 
Com ela, você pode criar e gerenciar suas tarefas diárias. você pode definir
o início e fim da tarefa e pode também verificar suas estatísticas e gerar metas
de crescimento pessoal, e acredito que o Tarefator pode ser o ideal para você!
